# JustDocs AI — Deployment Guide
# Read this top to bottom. Takes about 20 minutes total.

=============================================================
WHAT YOU NEED (all free accounts)
=============================================================
1. GitHub account (you have this)
2. Anthropic account + API key → https://console.anthropic.com
3. Cloudflare account (free) → https://cloudflare.com
4. Stripe account (free) → https://stripe.com

=============================================================
STEP 1 — GET YOUR CLAUDE API KEY
=============================================================
1. Go to https://console.anthropic.com
2. Sign up or log in
3. Click "API Keys" in the left sidebar
4. Click "Create Key" — name it "JustDocs"
5. COPY the key (starts with sk-ant-...) — save it somewhere safe
6. Add $5–10 in credits to start (Settings → Billing)
   At average usage, $5 = roughly 500–800 documents generated

=============================================================
STEP 2 — DEPLOY THE CLOUDFLARE WORKER (protects your API key)
=============================================================
1. Go to https://workers.cloudflare.com and sign up free
2. Click "Create Application" → "Create Worker"
3. Name it: justdocs-api
4. Delete all the default code in the editor
5. Paste the entire contents of worker.js into the editor
6. Click "Save and Deploy"
7. Copy your worker URL — it looks like:
   https://justdocs-api.YOURNAME.workers.dev

Now add your secret API key:
8. Go to your worker → Settings → Variables
9. Under "Environment Variables" click "Add variable"
10. Variable name: ANTHROPIC_API_KEY
11. Value: paste your Claude API key (sk-ant-...)
12. Check "Encrypt" box — this keeps it secret
13. Click Save

=============================================================
STEP 3 — SET UP STRIPE PAYMENT LINK
=============================================================
1. Go to https://stripe.com and create a free account
2. Go to "Payment Links" in the dashboard
3. Click "New payment link"
4. Add a product:
   - Name: "Legal Document — AI Drafted"
   - Price: $3.99 (or whatever you want)
5. Under "After payment" set success URL to:
   https://YOURUSERNAME.github.io/justdocs/?paid=true
6. Click Create — copy the payment link URL
   It looks like: https://buy.stripe.com/xxxxxxxxxxxxx

=============================================================
STEP 4 — UPDATE index.html WITH YOUR URLS
=============================================================
Open index.html and find these two lines near the top of the script:

  const WORKER_URL = 'YOUR_CLOUDFLARE_WORKER_URL_HERE';

Replace with your actual worker URL:
  const WORKER_URL = 'https://justdocs-api.YOURNAME.workers.dev';

Then find the pay button:
  <a href="YOUR_STRIPE_PAYMENT_LINK_HERE" ...>

Replace with your Stripe link:
  <a href="https://buy.stripe.com/xxxxxxxxxxxxx" ...>

Also update the ALLOWED_ORIGIN in worker.js:
  const ALLOWED_ORIGIN = 'https://YOURUSERNAME.github.io';
Then re-paste into Cloudflare and save.

=============================================================
STEP 5 — PUSH TO GITHUB PAGES
=============================================================
Option A — GitHub Website (no command line needed):
1. Go to github.com → "New repository"
2. Name it: justdocs (or whatever you want)
3. Make it Public
4. Click "uploading an existing file"
5. Drag and drop index.html into the upload area
6. Click "Commit changes"
7. Go to Settings → Pages
8. Under "Source" select "Deploy from a branch"
9. Branch: main, folder: / (root)
10. Click Save
11. Your site is live at:
    https://YOURUSERNAME.github.io/justdocs/

Option B — Git command line:
  git init
  git add index.html
  git commit -m "Launch JustDocs AI"
  git remote add origin https://github.com/YOURUSERNAME/justdocs.git
  git push -u origin main
  (then enable Pages in repo Settings)

=============================================================
STEP 6 — TEST IT
=============================================================
1. Visit your GitHub Pages URL
2. Select "Demand Letter"
3. Fill in the form with fake info
4. On the payment step — the app is currently in DEV MODE
   (it skips payment and generates the doc directly for testing)
5. Confirm the document generates correctly
6. Once confirmed, your Stripe link is already in place for
   real customers

=============================================================
STEP 7 — TAKING REAL PAYMENTS
=============================================================
The current flow:
- User clicks "Pay with Stripe"
- Stripe opens in new tab (or same tab)
- After payment, Stripe redirects to your site with ?paid=true
- The app detects ?paid=true and auto-generates the document

For a tighter flow (no tab switching) upgrade later to:
- Stripe Checkout embedded on your page
- Webhook that confirms payment server-side
(Claude can build this when you're ready)
